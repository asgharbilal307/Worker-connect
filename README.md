WorkerConnect is a modern web platform that connects customers with skilled workers such as electricians, plumbers, carpenters, and painters.
This repository contains the frontend application, built using Vite, React (JavaScript), Tailwind CSS, and lucide-react icons.

The frontend provides interfaces for:
✔️ Customers (browse workers, post jobs, manage bids)
✔️ Workers (view available jobs, place bids, manage profile)
✔️ Role selection and clean UI based on the Deliverable 1 specificatio
Customer Functionality:
Browse skilled workers by specialty, rating, and availability
Post custom jobs with:
title
category
location
budget
deadline
description
View and manage posted jobs
Compare bids submitted by workers
Direct hire & booking workflow
Worker Functionality:
Browse available job listings
Place bids with custom amount, proposal, and estimated time
Manage worker profile (bio, rate, experience, specialty, location)
Track submitted bids
General System Features:
Tailwind CSS UI based on modern marketplace apps
lucide-react icons for cleaner UI
Role selection screen (Customer / Worker)
Clean dashboard layout for each role
